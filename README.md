# kotlin-evaluation

## Description
Simulation simple d’un système bancaire en Kotlin.
Le projet montre comment créer un compte, effectuer des opérations et gérer un menu interactif en console.
## Contenu
Partie 1 : Interface + création du compte  
Affichage d’un écran d’accueil, choix du type de compte (debit, credit, checking).

Partie 2 : Opérations bancaires  
Fonctions principales :
withdraw (retrait générique)
debitWithdraw (retrait sécurisé)
deposit (dépôt générique)
creditDeposit (règles spécifiques au crédit)

Partie 3 : Gestion du compte
Fonction transfer(mode) pour router vers la bonne opération.
Menu principal en boucle (consulter solde, retirer, déposer, fermer l’app).
